## Description - EN:

This is a free, inofficial ORF TVThek App.

This app currently supports this features:
- Overview (Highlight, MostView, News)
- Programs (Show all available programs and mark them as favorite)
- Livestreams

## Description - DE:

Das ist eine gratis, inoffizelle ORF TVThek App.

Die App unterstützt derzeit folgende Featues:
- Übersicht (Aktuelle, Highlights, Meist Gesehen)
- Programme (Alle derzeit verfügbaren Programme anzeigen und diese als Favoriten markieren)
- Livestreams
